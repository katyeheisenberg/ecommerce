# E-commerce

## Description

It's an e-commerce page made with the help of React, Tailwind, Redux.

### How to use it?
Install modules
```
yarn install
```

Run a project in development mode
```
yarn run dev
```

Run server only
```
yarn run watch:server
```

Run client only
```
yarn run watch:client
```

Build a project
```
yarn run build:prod
```
